openkit-server
===============

After downloading build required gems with:
```
	$ bundle install
```


Start delayed _ job locally with:
```
	$ rake jobs:work
```

See a list of all rake-able jobs:
```
	$ bundle exec rake -T
```

Start server with:
```
	$ rails server
```
